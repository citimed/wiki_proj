# 🗄️ Inventory - приложени для ТСД(теримал сбора данных).

## 📝 Назначение (Бизнес-логика)
Приложение предназначено для проведения переучета в аптеках с использованием ТСД(терминал сбора данных). Таким образом уменьщая временные затраты и снижая вероятность ошибок для оперативного получения актуальных остатков аптеки. Использовался фреймворк Flutter
