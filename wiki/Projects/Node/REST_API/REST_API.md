# 🗄️ Основной REST сервис, который содержит в себе такие блоки:

## 📝 Назначение (Бизнес-логика)
 * Обработка заказов с агрегаторов Tabletki.ua и Liki24 с отправкой данных на аптеку
 * Обработка перемещений между РЦ и аптеками - синхронизация остатков
 * Обеспечение проверки остатков по сети с возможностью удаленной продажи(на другой аптеке) или заказа с РЦ
 * Синхронизацию чеков между ЦБД и аптекаим
 * Обеспечение функционала обработки прихода товара на аптеку силами фармацевтов
