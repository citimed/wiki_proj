# 🗄️ Метаданные Баз Данных

Добро пожаловать в единую базу знаний структуры баз данных проекта. Здесь описывается логика хранимых процедур, структура таблиц и их взаимосвязи для СУБД MS SQL Server 2019.

## 🏛️ Перечень Баз Данных

* [[MAIN_DB]] — Главная база данных системы (Core-логика).
* [[AP_DB]] — Аптечная база данных (Специфика розничного направления).
* **`[[AN_DB]]`** — База данных аналитики (OLAP-хранилище, исторические данные).
